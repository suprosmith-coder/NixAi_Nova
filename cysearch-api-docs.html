CYSEARCH API
Search and AI summarization for Cyanix users
v1.0 -- Supabase Edge
Base URL
https://tdbgpvscwaysndrloltl.supabase.co/functions/v1/cysearch-api

All requests are POST. The endpoint is selected via the "endpoint" field in the JSON body.

Authentication

Every request requires a valid Cyanix AI session token in the Authorization header. Only authenticated Cyanix users can access this API.

Required headers
Authorization: Bearer <supabase_access_token>
apikey:         <supabase_anon_key>
Content-Type:   application/json
Getting your token (from Cyanix app.js)
const { data: { session } } = await _sb.auth.getSession();
const token = session.access_token;
Endpoints
POST /cysearch-api endpoint: "search"

Searches DuckDuckGo and returns ranked results with an instant answer abstract.

Request body
FieldTypeRequiredDescription
endpointstringrequiredMust be "search"
querystringrequiredSearch query, 2--300 characters
Example request
fetch('https://tdbgpvscwaysndrloltl.supabase.co/functions/v1/cysearch-api', {
  method: 'POST',
  headers: {
    'Content-Type':  'application/json',
    'Authorization': `Bearer ${token}`,
    'apikey':        '<anon_key>',
  },
  body: JSON.stringify({
    endpoint: 'search',
    query:    'Supabase edge functions tutorial',
  }),
})
Response shape
{
  "query":       "Supabase edge functions tutorial",
  "abstract":    "Supabase Edge Functions are server-side...",  // may be empty
  "abstractUrl": "https://supabase.com/docs/...",
  "results": [
    {
      "title":   "Edge Functions | Supabase Docs",
      "snippet": "Deploy globally distributed TypeScript functions...",
      "url":     "https://supabase.com/docs/guides/functions"
    }
    // ... up to 6 results
  ],
  "relatedTopics": [
    { "text": "Deno Deploy", "url": "https://..." }
  ],
  "source": "duckduckgo",
  "ms":     842   // response time in milliseconds
}
POST /cysearch-api endpoint: "summarize"

Passes search results to Groq Llama 3.1 8B and returns a concise summary + key points. Typically chained after a search call.

Request body
FieldTypeRequiredDescription
endpointstringrequiredMust be "summarize"
querystringrequiredThe original search query (used as context)
resultsarrayrequiredArray of result objects from search (up to 5 used)
Example -- chaining search + summarize
// 1. Search
const searchRes = await cyApiCall('search', { query: 'black holes explained' });

// 2. Summarize the results
const summary = await cyApiCall('summarize', {
  query:   'black holes explained',
  results: searchRes.results,
});
Response shape
{
  "query":     "black holes explained",
  "summary":   "Black holes are regions of spacetime where gravity is so strong...",
  "keyPoints": [
    "Formed from collapsed massive stars",
    "Event horizon marks the point of no return",
    "Detected via gravitational waves and X-ray emissions"
  ],
  "ms": 1243
}
Error Codes
HTTPcodeMeaning
401 UNAUTHORIZED Missing or invalid JWT. Sign in to Cyanix AI first.
400 MISSING_QUERY The query field is missing or too short.
400 QUERY_TOO_LONG Query exceeds 300 characters.
400 MISSING_RESULTS The results array is missing or empty (summarize only).
400 UNKNOWN_ENDPOINT The endpoint field doesn't match any known endpoint.
502 SEARCH_ERROR DuckDuckGo upstream failure. Retry.
502 SUMMARIZE_ERROR Groq API failure. Retry.
Error response shape
{ "error": "human readable message", "code": "MACHINE_CODE" }
